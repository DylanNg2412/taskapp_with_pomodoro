# 🍅 Tomato Task

A Flutter-based task management application integrated with the Pomodoro Technique to help you stay focused and productive.

## 📝 About

Tomato Task combines traditional task management with the Pomodoro Technique, allowing you to organize your tasks and work on them in focused time intervals. Stay productive while maintaining a healthy work-rest balance.

## ✨ Features

- 📋 **Task Management** - Create, edit, and organize your tasks
- ⏰ **Pomodoro Timer** - Built-in Pomodoro timer for focused work sessions
- 📊 **Progress Tracking** - Monitor your productivity and completed tasks
- 🎨 **Clean UI** - Intuitive and user-friendly interface
- 💾 **Local Storage** - Your tasks are saved locally on your device
- 🌍 **Cross-Platform** - Works on Android, iOS, Web, Windows, macOS, and Linux

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (3.0.0 or higher)
- Dart SDK (3.0.0 or higher)
- Android Studio / Xcode / VS Code
- An emulator or physical device

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/DylanNg2412/taskapp_with_pomodoro.git
   cd taskapp_with_pomodoro
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```

## 🎯 Usage

1. **Add a Task** - Click the add button to create a new task
2. **Start Pomodoro** - Select a task and start the Pomodoro timer
3. **Focus Time** - Work on your task during the 25-minute focus session
4. **Take Breaks** - Enjoy short breaks between Pomodoros
5. **Track Progress** - Mark tasks as complete and review your productivity
